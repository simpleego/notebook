# React 개발환경 구축하기 

--- 
## Node.JS 설치
> Node 사이트 방문하여 다운로드 후 설치
## yarn 설치
>npm install -g yarn 
## 설치 버전 확인
> node --version    또는  node --v
> npm --version  또는  npm --v
## React 프로젝트 생성
> **yarn create react-app 프로젝트이름**  
> yarn create react-app my-app
